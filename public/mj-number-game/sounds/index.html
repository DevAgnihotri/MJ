MJ Sound Effect Extractor

Create sound effects for your THRILLGUESSER game

This tool helps you extract short clips from Michael Jackson songs to use as sound effects in your game. Here's how it works:

  1. The page will load audio files from your /audio folder
  2. Click "Extract Sound Effects" to generate clips for each game sound
  3. Adjust the start time and duration using the sliders
  4. Click "Play" to preview each sound
  5. When you're satisfied, click "Download" to save the sound file

The sounds will be saved to your downloads folder, and then you can move them to the game's sounds folder.
