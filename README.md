# Outlier AI Hackathon Project

Created by Dev Agnihotri

This project includes a collection of interactive mini-games and features, including:

- MJ Music Player
- AI Roller
- Blocks Game
- Colorful Memory Game
- Image Gallery
- Mafia Game
- Moonwalk Experience
- Pinball Game
- PMD-Clock (Perpetual MJ Dance Clock)
- Rock Paper Scissors
- Speed Analysis Tool
